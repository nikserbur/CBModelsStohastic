## Gradient Boosting –¥–ª—è –≤—ã–±–æ—Ä–∞ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–π

–ê–¥–∞–ø—Ç–∏–≤–Ω—ã–π –≤—ã–±–æ—Ä –æ–ø—Ç–∏–º–∞–ª—å–Ω–æ–≥–æ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç —á–µ—Ä–µ–∑ –≥—Ä–∞–¥–∏–µ–Ω—Ç–Ω—ã–π –±—É—Å—Ç–∏–Ω–≥ —Å —Ñ—É–Ω–∫—Ü–∏–µ–π –ø–æ—Ç–µ—Ä—å `L(Œ∏) = -Œ£·µ¢‚Çå‚ÇÅ‚Åø log p(y·µ¢|x·µ¢, Œ∏)`, –æ–±–Ω–æ–≤–ª–µ–Ω–∏–µ–º –≤–µ—Å–æ–≤ `w^(t+1) = w^(t) √ó exp(-Œ∑ √ó ‚àáL(Œ∏^(t)))` –∏ —Ñ–∏–Ω–∞–ª—å–Ω–æ–π –º–æ–¥–µ–ª—å—é `p(y|x) = Œ£‚Çñ‚Çå‚ÇÅ·¥∑ w‚Çñ √ó p‚Çñ(y|x, Œ∏‚Çñ)`, –≥–¥–µ K = {Normal, t-dist, Œ±-stable, GED}.
